# Changelog

All notable public source changes for ScanValue are documented in this file.

## 0.0.1 - 2026-06-10

### Added

- Client-side scan-based scrap value labels for Lethal Company.
- Item names shown alongside scanned scrap values.
- Value-based label and outline colors with configurable value bands.
- Pooled world-space label rendering to keep scan UI overhead low.
- Optional scanned scrap outline highlighting.
- Vanilla scan integration that keeps the original total value behavior while replacing scrap item scan boxes.
- Runtime config reload support through BepInEx.
- English and Chinese config descriptions, with LC Chinese Project detection that does not depend on a fixed localization mod version.
